## Overview

A plain collection of Swift extensions to help speed up development.

## Requirements
* iOS 11
* Swift 5.0

## Installation

PlainExtension is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'PlainExtension'
```
